FOTOCOPIADORA

Fotocopiadora es una aplicación web que ayuda a los usuarios a editar recibos de los servicios públicos con los que cuentan para realizar trámites. Las opciones que incluyen son la edición de la imagen en: opacidad, saturación y exposición, incluye tambien tres filtros para modificar la imagen. Pueden subir imagenes e imprimirlas facilmente desde la aplicación web.

Esta aplicación web esta diseñada como sitio responsive asi que se adapta a los distintos dispositivos como tablets, celulares y computadoras.
Lenguajes utilizados: HTML 5, CSS, JAVA Y JQUERY.
